What it does

A free, offline CLI that tells you whether an n8n workflow export will actually import — before you buy, sell, or ship it. It flags leaked credential secrets, dangling credential references, missing community nodes, version drift, deprecated syntax, and broken connections. Verdicts were arbitrated against a live n8n import API across 20 real workflows with 0 mismatches.

The free CLI diagnoses. FlowProof Pro repairs the workflow into an import-ready file and issues a content-hashed "FlowProof Checked" certificate + client handover: fp.akuchis.com.
